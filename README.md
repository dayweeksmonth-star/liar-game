# 라이어 게임 AI

여러 명이 함께 접속해서 즐기는 단어 기반 라이어 게임입니다. 혼자서도 연습할 수 있도록 AI 참가자 모드를 포함했습니다.

## 주요 기능

### 여럿이 하기

- 닉네임 입력
- 방 생성 / 방 코드 입장
- 같은 방 참가자 실시간 동기화
- 방장만 게임 시작 가능
- 단어 주제 선택
  - 쉬움
  - 직장/공공기관
  - 어려움
- 시민에게만 단어 공개
- 라이어에게는 단어 비공개
- 참가자별 힌트 제출
- 투표로 라이어 지목
- 라이어가 잡히면 마지막 단어 추측 기회 제공
- 점수판 제공
- 라운드 재시작 가능

### 혼자 AI랑 하기

- AI 참가자 3명과 혼자 플레이
- 사용자가 시민 또는 라이어로 랜덤 배정
- AI가 자동으로 힌트 작성
- AI가 힌트의 구체성/애매함을 기준으로 라이어 추리
- AI 투표 기능
- 라이어 최종 추측 기능

> 현재 AI 기능은 외부 API 없이 작동하는 룰 기반 AI입니다. 나중에 OpenAI API 등을 연결하면 더 자연스러운 힌트와 추리가 가능해집니다.

## 설치 및 실행

```bash
npm install
npm start
```

실행 후 브라우저에서 접속합니다.

```text
http://localhost:3000
```

## 여러 명이 접속하는 방법

같은 와이파이 또는 같은 사무실 네트워크에 있는 사람들은 서버를 실행한 PC의 내부 IP로 접속할 수 있습니다.

예시:

```text
http://192.168.0.10:3000
```

외부 인터넷에서 접속하려면 ngrok, Cloudflare Tunnel, Render, Railway 같은 배포 방식이 필요합니다.

## 게임 규칙

1. 방장이 게임을 시작합니다.
2. 시민들은 같은 단어를 받습니다.
3. 라이어는 단어를 모릅니다.
4. 각자 단어를 직접 말하지 않고 힌트를 제출합니다.
5. 모든 힌트가 제출되면 라이어로 의심되는 사람에게 투표합니다.
6. 라이어가 잡히면 라이어에게 마지막 단어 추측 기회가 주어집니다.
7. 라이어가 단어를 맞히면 라이어 승리, 틀리면 시민 승리입니다.
8. 라이어를 못 찾으면 라이어 승리입니다.
